# Basic User Workflow

- Work in right-of-way (street), safer faster lower risk
- Goal is to create a shared space with visibility for all project proponents to make project completion easier.

- Drawing work space (work areas/polygons)

Flow:

- Create a project, rename
- Create a work area, rename, should display:
  - calculated area in m2
  - complexity (inferred from pop density)
  - duration (??)
  - coverage ?
- Clicking on Work area should open side menu that displays:
  - project name
  - summary
  - upload records

Hierarchy:

- Work areas => assets/geometry => records

- Share link to Embridge (for example) when requesting records for the area. Link should include viewstate of the user when generating the link

Their legend:

- Water (blue)
- Wastewater (purple)
- Storm (green)
- Gas (yellow)
- Electric (red)
